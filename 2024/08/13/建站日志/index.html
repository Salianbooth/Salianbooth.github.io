建站日志

Author: Salianbooth Date: August 13, 2024  17:36:53 Category: Log

使用 hexo 建站后，改用了新的主题，但存在许多问题，后续将不断修复更新，本文将记录主要的更新内容。

Before

在使用Chic主题之前，使用的是vivia主题，站点样式均为自定义。

Display

2024-8-13

在学长miaoi的见一下，我选择了hexo来进行建站，并且选择了Chic主题，这是一款简洁明了且功能强大的主题，接下来我就建站过程进行简要说明。

Install

  • 安装hexo
    在确保安装好 node.js 和 git 之后执行以下指令：
  • 下载Chic

参考 Chic 的github地址 https://github.com/Siricee/hexo-theme-Chic 给出的方式可以简单的安装

Configuration

  • 在myBlog下的_config.yml可以进行theme和GitHub的绑定
  • 在theme下的Chic文件下的 config.yml 可以进行个性化的处理
    • 更改导航名称
    • 更改网站图标
    • 更改个性签名和头像
    • 显示GitHub地址

Deploy

  • 创建GitHub个人仓库
    • 要以个人用户名+github.io结尾
  • 生成ssh添加到Github
  • 进行部署
    • 这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。
  • 进行部署

Fix

最开始遇到了插入的图片不显示的问题，在查阅相关资料后得到解决，一下是解决方案
  • 在chic目录下面找到head.ejs或者是有相同后缀名的文件
    • 具体路径在 Chic/layout/____partial/head.ejs
    • 在文件中添加meta信息即可
